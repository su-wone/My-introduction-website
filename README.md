# My Introduction Homepage Renewal 🧑‍💻
자기소개 홈페이지를 리뉴얼한 정적 웹사이트 프로젝트입니다.
사용자 경험(UX)을 개선하고, 시각적으로 깔끔한 레이아웃과 콘텐츠 구조를 목표로 제작했습니다.
​

🔗 프로젝트 개요
프로젝트명: My Introduction Homepage Renewal
​

주제: 자기소개·취미·여행·연락처를 한 번에 볼 수 있는 개인 홈페이지
​

목적: 웹 디자인 및 프론트엔드 기초 역량 강화, 포트폴리오용 자기소개 페이지 제작
​

형태: 정적 웹사이트 (Static Website, HTML/CSS 기반)
​

개발자: 강수원

🛠 사용 기술
HTML5

CSS3

미디어 요소 활용 (Image / Audio / Video)

Flexbox 기반 레이아웃
​

📂 프로젝트 구조
text
My-Introduction-Homepage-Renewal/
├── index.html              # 메인 페이지 (소개 + 네비게이션)
├── hobby.html              # 취미 소개 페이지
├── travel-Philippines.html # 필리핀 여행 소개 페이지
├── contact.html            # 연락처 / SNS 페이지
├── style.css               # 공통 스타일 시트
│
├── images/                 # 이미지 리소스
│   ├── profile.jpg
│   ├── hobby_soccer.jpg
│   ├── philippines1.jpg
│   └── ...
│
├── media/                  # 미디어 파일
│   ├── intro_bgm.mp3
│   └── intro_video.mp4
│
└── README.md               # 프로젝트 설명 문서
🧾 주요 기능 및 콘텐츠
자기소개: 이름, 생년월일, 거주지, 학교/전공, MBTI 등 기본 정보 표 형식 제공
​

취미 소개: 축구, 야구, 책 읽기 등 여러 취미를 카드 형식으로 정리
​

여행 기록: 필리핀 여행 사진과 간단한 후기, 추억 정리
​

연락처: 이메일 및 SNS 링크 제공으로 소통 채널 안내
​

▶ 실행 방법
프로젝트 클론 또는 다운로드

프로젝트 폴더로 이동

bash
cd My-Introduction-Homepage-Renewal
index.html 파일을 더블 클릭하여 브라우저에서 실행

📌 기타
포트폴리오 및 자기소개 용도로 언제든지 구조와 디자인을 확장·수정할 수 있습니다.

추후 JavaScript를 추가해 인터랙션과 애니메이션을 넣어 발전시킬 수 있습니다.
